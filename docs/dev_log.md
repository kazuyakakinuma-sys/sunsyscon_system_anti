## プロジェクト概要（再掲）

楽天銀行の入金履歴CSVを読み込み、kintone上の顧客データを照合・更新するためのCSVを生成するWebアプリ。  
kintoneへの書き込みはCSVインポートで行うため、APIは使用しない。

---

## 確定した仕様

### 入力①：楽天銀行CSV
- 文字コード：**Shift-JIS**（読み込み時にUTF-8変換が必要）
- 列構成：A=取引日(YYYYMMDD) / B=入出金(円) / **C=累計(円)** / D=入出金先内容（振込名義）
  - ※ C列（累計）は当初仕様書に未記載だったが、実データ検証（2026-03-03）で判明。
- 処理対象：B列がプラス（入金）のレコードのみ
- 特徴：
  - 振込名義は全角カタカナ。半濁点(゛)が独立文字として存在する場合がある
  - 全角スペースで姓と名が区切られている場合がある
  - 電話番号・氏名以外の情報が混在する場合がある
  - 出金行には銀行名・口座情報・長文が含まれる
  - 「依頼人名：」を含む行は自社出金（処理対象外）

### 入力②：kintone各アプリCSV（6アプリ、STEP 2でアップロード）
1. 顧客データ
2. 契約回線情報
3. 遅延情報
4. uCloudlink SIMBANK１
5. WEBキントーン連携 NEW新規申込
6. Wi-Fi回線及びクラウドデータSIM情報

> ※ 現状はワークフローが属人化しているため6アプリすべてを対象とする。
> ※ 開発・整理の過程で不要と判断されたCSVがあれば都度指摘する。

### 出力：kintoneインポート用CSV
| 列 | 列名 | 内容 |
|----|------|------|
| A | レコード番号 | kintoneのレコード番号 |
| B | 顧客ID | kintone上の顧客ID |
| C | 氏名 | 顧客氏名 |
| D | フリガナ | 顧客氏名のフリガナ |
| E | 契約月数 | 契約経過月数 |
| F | 過不足 | 入金額と請求額の差分 |
| G | 支払ステータス | 支払い状況 |
| H | 契約状況ステータス | 契約の状態 |
| I | 遅延認証 | 遅延フラグ・認証情報 |

## 実装済み：Phase 01 楽天銀行CSVの正規化・フリガナ変換

**実行スクリプト：** `scripts/phase01_process_bank_csv.py`
**ルール定義ファイル：** `scripts/rules_furigana.py` (変換マップ・ホワイトリスト・除外リストなどを定義)

### 確定した変換ルール（順番通りに適用）

#### ① 除外パターンチェック（先頭で判定、空欄を返す）

個人顧客でない振込元は、**入金であっても**照合対象外として空欄にする（業務上の仕様）。

| 登録パターン | 対象 |
|-------------|------|
| `依頼人名：` | 自社出金・振替行 |
| `ＣＳＳ（ＭＦＲＬ` | CSS/MFRL賃料管理費 |
| `ロボツトペイメント` | ロボットペイメント（決済代行会社） |
| `コクリツケンキユウカイハツ` | 国立研究開発法人医薬基盤健康 |
| `ラクテンカ−ト゛` | 楽天カードサービス |
| `ニホンセイメイホケン` | 日本生命保険 |

> 追加する場合は `EXCLUDED_PATTERNS` リストの末尾コメント部分に1行追記するだけ。

#### ② 半濁点の結合
- 独立した `゛`（U+309B）を直前の文字と結合
  - 例：`イツ゛マ` → `イヅマ`、`オカ゛ワ` → `オガワ`

#### ③ 拗音変換（**全角スペースで単語分割してから各単語に適用**）
- 境界をまたいだ誤変換防止が目的（例：`サカグチ　ユウイチ` の `チユ` が `チュ` になるのを防ぐ）
- 変換マップ：キヤ→キャ / シヤ→シャ / チヤ→チャ / など11パターン × ヤ/ユ/ヨ行
- **ホワイトリスト保護**：プレースホルダー方式で保護対象を一時退避してから変換
  - 保護対象：アキヤマ、ニシヤマ、ミヤモト、ミヤザキ 等（`PROTECTED_SUBSTRINGS` に追記で拡張可能）
  - ミユ、ミユキ も保護対象（名前として大きいユを維持）

#### ④ 不要文字削除（カタカナのみ残す）
- 漢字・数字・記号・英字・ハイフン・スペースをすべて削除

#### ⑤ 促音変換（不使用）
- 名前での誤変換が多発するため採用しない

### 変換例

| 変換前（銀行CSV） | 変換後 | 備考 |
|-----------------|--------|------|
| `イツ゛マ　ヒロユキ` | `イヅマヒロユキ` | 半濁点結合 |
| `ナカジヨウ　マサノリ` | `ナカジョウマサノリ` | ジヨ→ジョ |
| `サイトウ　シヨウタ` | `サイトウショウタ` | シヨ→ショ |
| `サカグチ　ユウイチ` | `サカグチユウイチ` | 境界またぎを防止 |
| `アキヤマ　マキ` | `アキヤママキ` | ホワイトリスト保護 |
| `クガ　アツミ　０８０８...` | `クガアツミ` | 電話番号削除 |
| `依頼人名：` 含む行 | `（空欄）` | 自社出金 |
| `振込手数料（内消費税 ￥20）` | `（空欄）` | 数字・記号のみ |
| `ニホンセイメイホケン` | `（空欄）` | 法人除外（入金でも対象外） |
| `カ）ロボツトペイメント` | `（空欄）` | 法人除外（入金でも対象外） |
